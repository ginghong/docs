## Goal

Maintain alignment with minimal effort using rituals and rules.

## Controls

- Drift thresholds and rebalancing
- Event-driven updates (narrative change)
- On-chain execution preferences

<Steps>
<Step title="Set drift limits">
  Choose a threshold (e.g., ±5–10%). Only rebalance when crossed.
</Step>

<Step title="Schedule reviews">
  Run a weekly scan and a deeper quarterly review.
</Step>

<Step title="Execute on-chain">
  Use self-custodial flows; confirm gas, slippage, and venue.
</Step>
</Steps>
